Script Installation

This script is a macro-script so it needs to be installed in a 3DS Max toolbar :

  • 1. Copy mcrender_v1.2.ms file anywhere on your hard disk
  • 2. Execute mcrender_v1.2.ms under 3DS Max [ MAXScript/Run Script... ]
  • 3. Toolbar installation :
    • a. [ Customize/Customize User Interface ] and select [ Toolbars ] window tab
    • b. If you don't want to create a new toolbar jump to c)
      Press [ New ] and enter a toolbar name. A floating toolbar is then created. You can dock this toolbar dragging it on interface edges.
    • c. In [ Category ] drop-down menu, select "XC_SCRIPT"
    • d. Drag and drop "Multi Camera Render" in an existing toolbar or new one you've just created
  • 4. The script is now ready to be used

Script Interface Help

1. [Target Path and Files] section

Target directory :

Set the path location for output rendered image files. Click on [..] to browse the disk.

Output file names :

Output rendered image names. Name is built from the name of current camera being rendering adding prefix and suffix provided by user (could be left blank).

Use the drop-down menu to select image format (TGA, JPG, BMP, TIF, PNG).

Log file :

Check this box and select a file name and a location to generate a basic rendering log file. Click on [..] to browse the disk.

[ Load.. ] :

To load from a file a previous "Multi Camera Render" set of parameters.

[ Save.. ] :

To save to a file current "Multi Camera Render" set of parameters.

[ More Options.. ] :

Open [ Render Scene ] dialog box to configure other parameters for rendering.
BE CAREFULL : do not press [ Render ] in this dialog box. You should close the window to return to "Multi Camera Render" interface.

2. [Cameras Selection] section

Scene cameras :

Current cameras in the scene (udpated each time the script is executed).

Cameras to render / Scripts :

Rendering list : current selection of cameras to be rendered and scripts to be executed.

[ >> ] and [ << ] :

Used to add or remove camera(s) to / from rendering list.

[ up ] and [ down ] :

Move up or down current selected item(s) in rendering list.

[ |< ] and [ >| ] :

Used to add or remove script(s) to / from rendering list.

S and L spinners :

Used to select which frame(s) to render for each camera. No effect on scripts.

3. [Rendering Sequence] section

Enable pre-rendering script :

Used to enable and select a script to be executed before each camera rendering.

Reverse cameras list :

When this option is checked, rendering sequence is from the last camera/script of rendering list to the first one, i.e down-top rendering. Default is top-down rendering.

4.[Render Cameras] button

Push this button to render selected cameras and execute selected scripts of rendering list. To cancel rendering, press [ESC].
