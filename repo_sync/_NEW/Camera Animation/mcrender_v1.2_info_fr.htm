Note Importante

Ce script est libre de droits et vous pouvez l'utiliser ou le distribuer sans aucune restriction. Cependant, si vous utilisez une partie du code dans un de vos propre scripts, n'oubliez pas de citer l'auteur d'origine en tant que contributeur.

Installation du script

Ce script est un macro-script et nécessite donc d'être installé au préalable dans une barre d'outils de 3DS Max :

  • 1. Copier le fichier mcrender_v1.2.ms n'importe où sur votre disque dur
  • 2. Exécuter mcrender_v1.2.ms sous 3DS Max [ MAXScript/Exécuter script...]
  • 3. Installation dans la barre d'outils :
    • a. [ Personnaliser/Personnaliser interface utilisateur ] et sélectionner l'onglet [ Barres d'outils ]
    • b. Si vous ne voulez pas créer une nouvelle barre d'outils rendez-vous directement en c)
      Sélectionner [ Nouveau ] et entrer un nom pour la barre d'outils. Une nouvelle barre d'outils flottante est alors créée. On peut la fixer en la glissant vers l'un des bords de l'interface
    • c. Dans le menu déroulant [ Catégorie ], sélectionner "XC_SCRIPT"
    • d. Tirer "Multi Object Modifier" dans une barre d'outils existante ou dans celle que vous venez de créer
  • 4. Le script est maintenant prêt à être utilisé

Aide pour l'interface

1. Section [Target Path and Files]

Target directory :

Défini le répertoire où seront sauvegardées les images rendues. Cliquez sur [..] pour explorer le disque.

Output file names :

Nom des images de sortie. Il est construit à partir du nom de la caméra en ajoutant le préfixe et le suffixe définis par l'utilisateur (ces 2 champs peuvent être vides).

Utiliser le menu déroulant pour sélectionner le format d'image (TGA, JPG, BMP, TIF, PNG).

Log file :

Sélectionner cette option pour générer un fichier de rapport de rendus. Cliquez sur [..] pour explorer le disque.

[ Load.. ] :

Charger les paramètres de l'interface "Multi Camera Render" depuis un fichier.

[ Save.. ] :

Sauver les paramètres courants de l'interface "Multi Camera Render" dans un fichier.

[ More Options.. ] :

Ouvre la boîte de dialogue [ Rendu scène ] qui permet d'accèder à des paramètres de rendu supplémentaires.
ATTENTION : ne pas appuyer sur [ Rendu ] dans cette boite de dialogue. Vous devez fermer cette fenêtre pour retourner à l'interface "Multi Camera Render".

2. Section [Cameras Selection]

Scene cameras :

Liste des caméras existantes dans la scène (mise à jour à chaque exécution du script).

Cameras to render / Scripts :

Liste des rendus : liste des caméras à rendre et des scripts à exécuter.

[ >> ] and [ << ] :

Ajouter ou supprimer une ou plusieurs caméras à/de la liste des rendus.

[ up ] and [ down ] :

Déplace vers le haut ou vers le bas un ou plusieurs items de la liste des rendus.

[ |< ] and [ >| ] :

Ajouter ou supprimer un ou plusieurs scripts à/de la liste des rendus.

Spinners S et L :

Permet de sélectionner les frames à rendre pour chaque caméra. Aucun effet sur les scripts.

3. Section [Rendering Sequence]

Enable pre-rendering script :

Activer cette option pour sélectionner un script qui sera exécuté avant chaque rendu de caméra.

Reverse cameras list :

Quand cette option est sélectionnée, la séquence des rendus s'exécute depuis le bas de la liste des rendus vers la haut. Par défaut, la séquence s'éxecute de haut en bas.

4. Bouton [Render Cameras]

Presser ce bouton pour rendre les caméras sélectionnées et exécuter les scripts de la liste des rendus. Pour interrompre le processus, appuyer sur [ESC].
